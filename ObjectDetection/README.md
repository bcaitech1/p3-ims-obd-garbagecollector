# How to use

### Check dataset
``` bash
python dataset.py --config [config]
```

### Check the model
``` bash
python model.py --config [config]
```

### Train
1. train.ipynb
2. change config 
3. run all


### Inference
``` bash
python inference.py --config_name [config]
```

### Ensemble
1. k-fold.ipynb
2. TTA.ipynb
3. ensemble_nms.ipynb
